# FranDev Dashboard

Executive-layer dashboards for Mainstream Boutique Franchising franchise development pipeline.

- **CEO Dashboard v1** — weekly pipeline snapshot for CEO
- **Project Status** — live status of the Katie integration phase 1 build

## Pages

- `index.html` — landing page with links
- `ceo-dashboard.html` — CEO pipeline glance
- `project-status.html` — internal project status

## Notes

Aggregate data only. Lead names, exact counts, and internal IDs are redacted for web viewing. Full detail lives in FranConnect. `robots.txt` and `<meta name="robots" content="noindex">` ask search engines not to index.

## Refresh cadence

Snapshot is refreshed manually each Friday EOD. v2 will auto-refresh via FranConnect API.
